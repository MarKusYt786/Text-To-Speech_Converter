# üß† Text-to-Speech Converter
The Text-to-Speech Converter, a Web Application is a user-friendly web-based platform that allows users to input text and hear it spoken in a natural-sounding voice. The application utilizes the Web Speech Synthesis API to synthesize speech from the user's input text, providing a seamless and interactive experience. The core features of the application include text input, voice selection, and speech control, making it an ideal tool for individuals who want to listen to text-based content.

## üöÄ Features
- **Text Input**: Users can input text into a textarea element, which is then synthesized into speech.
- **Voice Selection**: Users can select from a variety of voices to customize the speech output.
- **Speech Control**: Users can control the speech output using buttons for speaking, pausing, and stopping speech.
- **Google Analytics Integration**: The application tracks website usage using Google Analytics.

## üì¶ Installation
To get started with the Text-to-Speech Web Application, follow these steps:
### Prerequisites
- A modern web browser with support for the Web Speech Synthesis API
- An internet connection for loading the Google Analytics library

### Installation
1. Clone the repository using Git: `git clone https://github.com/your-username/your-repo-name.git`
2. Open the `index.html` file in a web browser to launch the application

### Running locally
1. Open the `index.html` file in a web browser to launch the application
2. The application will load the Google Analytics library and initialize the text-to-speech functionality
